### Requirements 
- [NodeJS](https://nodejs.org/en/)

### Installation
- ```git clone https://github.com/ChristofferHolmesland/InteraktivUndervisning.git```
- ```cd InteraktivUndervisning/App```
- ```npm install```

### Start Commands
- ```node server/main.js```
